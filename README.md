## 💡 Dashboard Features

- 👤 Sales by Executive
- 📅 5-day sales tracking
- 🎯 Target vs. Achieved Percentage
- 📉 Difference from Target (underperformance)
- 📊 Interactive visuals with slicers and pivot charts
- 💻 Macro-enabled for reset filters or automation
